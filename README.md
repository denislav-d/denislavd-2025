# Denislav Dimitrov Personal Portfolio 2025

A modern, interactive portfolio website showcasing creative technology projects with smooth animations and immersive experiences.

## Tech Stack

- **Framework:** Next.js 15 (App Router) with React 19
- **Language:** TypeScript
- **Styling:** TailwindCSS 4
- **Animation:** GSAP with ScrollTrigger
- **3D Graphics:** Three.js with custom GLSL shaders
- **Smooth Scrolling:** Lenis
- **Content:** MDX with custom rendering
